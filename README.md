# RAG_ChatBot
RAG-based chatbot for domain-specific Q&amp;A (e.g., Finance Assistant). Upload PDFs/CSVs/text, and it answers only from those sources using semantic retrieval + LLMs. Built with LangChain, FAISS/ChromaDB, Hugging Face, FastAPI, Streamlit, and Docker. Deployable on Hugging Face Spaces or AWS.
